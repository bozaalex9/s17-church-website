# S17 Church Website — Optimized Code Foundation

This package contains the approved S17 website code foundation with optimization/audit updates applied.

## Governance

All design and implementation work is governed by the [S17 Website Governance](./docs/governance/README.md) documents. The current Website Constitution supersedes every prior constitution and any guidance derived from one.

## Included

- Approved global layout
- Threshold navigation
- Footer
- Homepage
- Visit
- Gather
- Grow
- Go
- About with Beliefs section
- Give page as Church Center bridge
- Design tokens
- Motion / Journey system
- Accessibility and performance optimizations

## Not included

- Separate Beliefs page
- Separate Sermons page
- Generosity route

## Production checklist

1. Install dependencies.
2. Run `npm run build`.
3. Replace placeholder image treatments with approved S17 photography.
4. Confirm Church Center Giving URL.
5. Confirm social and YouTube links.
6. Run Lighthouse on production build.
