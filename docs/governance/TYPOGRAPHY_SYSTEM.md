# S17 Editorial Typography System

## Voice

Helvetica Neue remains the sole type family. Confidence comes from scale, weight, rhythm, hierarchy, and disciplined copy—not novelty fonts or decorative effects.

## Hierarchy

- Give each major scene one dominant, poster-like headline.
- Use bold weight, tight line-height, and slight negative tracking for oversized display text.
- Let large headlines proclaim; let supporting copy explain calmly.
- Keep labels and utility text compact, legible, and clearly subordinate.
- Use generous space after a headline so comprehension precedes detail or action.
- Avoid several similarly loud text elements in one viewport.

## Responsive behavior

Mobile headlines remain emotionally substantial but must avoid accidental orphaning, horizontal overflow, or unreadable compression. Text must remain selectable, zoomable, semantic, and stable while being read.

## Proclamation revision

Typography should not default to polite section-title scale. Worship, welcome, mission, testimony, and invitation may use the largest editorial scale the composition can sustain clearly.
